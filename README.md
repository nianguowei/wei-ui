# wei-ui
基于VUE，模仿element-ui，手动写的常用组件库，摘取的核心代码，主要帮助学习VUE以及element-ui组件的实现原理

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
